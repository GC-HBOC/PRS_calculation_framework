PRS models are briefly explained [here](https://github.com/GC-HBOC/PRS_calculation_framework/tree/main/mpileup_vcf).
`BRIDGES_306_CFBOC` correspond to the [BRIDGES 306](https://github.com/CCGE-BOADICEA/SHARE-PRScalculation/blob/main/PRSmodels_CanRisk/BRIDGES_306_PRS.prs) PRS model, but with default dosage imputation for 9 loci, which were previously identified as allele frequency outliers when genotyped via NGS \[[Baumann et al. 2024\](https://doi.org/10.1038/s41431-024-01647-2).

Note that suffixes `*_hg19.tsv` or `_hg38.tsv` have to match the reference genome used for genotyping.

Are you missing a PRS model or have found an error? Please do not hesitate to create an [issue](https://github.com/GC-HBOC/PRS_calculation_framework/issues).
